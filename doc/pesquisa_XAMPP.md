## oque é o XAMPP

O XAMPP é um pacote de softwares gratuitos que cria um servidor web local no seu computador. Ele permite testar e desenvolver sites e sistemas (como WordPress, lojas virtuais ou aplicações em PHP) direto na sua máquina, sem precisar de internet ou de um servidor pago.
O conjunto Apache, MySQL/MariaDB, PHP e phpMyAdmin forma a clássica pilha de desenvolvimento web. Eles operam em conjunto: o servidor (Apache) recebe as requisições, processa a lógica da aplicação (PHP) e gerencia os dados salvos (MySQL/MariaDB), enquanto o phpMyAdmin fornece a interface gráfica para controlar o banco.
Os componentes são as peças, dispositivos ou módulos individuais que constituem um sistema maior (seja um circuito eletrônico, um computador ou um projeto mecânico). Eles trabalham em conjunto para que o equipamento realize sua função específica. 


## como se instala
A instalação e configuração básica de um sistema operacional (como Windows ou Linux) consiste em preparar o disco, instalar os arquivos essenciais, criar uma conta de usuário e atualizar o sistema. Este processo garante que o seu computador esteja pronto para receber os programas que você precisa utilizar.
